보안이 설정된 보안영역(폴더)와 일반영역(폴더)의 위치를 입력받아
각 영역에 문서(doc, docx, ppt, pptx, xls, xlsx, hwp, hwpx, txt 등)은 꼭 포함하고, 시스템 데이터, 시스템 환경 데이터 등 중요 데이터 등의 데이터를 생성하고
각 영역에 대해 랜섬웨어 침해여부를 측정하여 걸과 데이터(.csv 파일)를 생성하는 powershell 관리자모드에 복붙하여 실행하기 위한 하나의 자동화 스크립트를 만들어줘
랜섬웨어 시물레이터는 기본 축으로는 RanSim(안전한 “모의 암호화/행위” 시나리오로 엔드포인트 보호(EDR/백신/ASR/폴더 보호)의 탐지·차단 반응을 빠르게 확인)으로 하고 보완 축으로 Atomic Red Team / Caldera(MITRE ATT&CK 전술·기술(TTP)로 랜섬웨어 전 단계(권한 상승, 파일권한 조작, 의심 스크립트/압축/유출 시도 등)를 폭넓게 재현), 복구 검증 축으로 백업/버전관리/스냅샷의 RPO/RTO 달성 여부까지 함께 검증(암호화가 일부 성공해도 얼마나 빨리/완전하게 복구되는지)으로 진행하고자 함
스크립트를 실행하면 가장 먼저 일반영역과 보안영역의 위치를 물어보고, 결과 데이터를 생성할 위치를 물어보고 다음 단계를 진행하며, 
랜섬웨어 침해여부 테스트 전에 각 시물레이터(RanSim, Atomic Red Team / Caldera 등)이 설치되어 있는지 확인하고 설치되어 있으면 다음 단계로 넘어가고, 설치되어 있지 않으면 각 시물레이터를 다운로드하여 설치하도록 하여야 함
